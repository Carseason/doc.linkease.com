## 沙箱模式

* 沙箱实际是多层OverlayFS，方便用来实验系统配置和程序，方便开发未完成的软件，但不保护Docker和硬盘的数据。

* 开启沙箱准备工作：
```
1. 准备一个U盘或者移动硬盘，方便装卸
2. 分区格式化成ext4
3. 在挂载点页面配置此分区，其中 “挂载点” 选项选择 “作为外部 overlay 使用”
4. 重启即可进入沙箱环境
```
![png](./picture/Sandbox.png)
